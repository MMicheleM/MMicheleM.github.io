Classes vs. IDs

One to one (ID) vs. one to many (class)

You are (hopefully!) the only one with your ID but there are a bunch of us in this class/cohort. Therefore if you wished to apply your style to yourself you'd select ID however if you wanted to share your fabulous style with all of us you'd want to select class.

They both:

  • Define a special case for an element

Syntax

In HTML you'll use similar syntax. It'll either be id = or it'll be class =

In CSS you'll being your ID with a hashtag # and your class with a fullstop .
